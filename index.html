Perguntas

1 - Quais os Comandos para onfigurar o usuário e e-mail no GIT?

2 - Qual a diferença entre git e github?

3 - O Que faz o comando commit?

4 - O que faz o comando push?


Respostas

1 - $ git config --global user.name "Mateus Castro"
$ git config --global user.email cbmateus1@gmail.com

2 - git é um software VCS local que permite aos desenvolvedores salvar snapshots de seus projetos ao longo do tempo. Geralmente é melhor para uso individual. GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões do gitHub para que possam ser usados colaborativamente.

3 - O comando git commit captura um instantâneo das mudanças preparadas do projeto no momento

4 - O comando push transfere commits do repositório local a um repositório remoto
